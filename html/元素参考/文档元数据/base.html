<base>: 文档根 URL 元素

  1. 多个<base>元素，只会使用第一个 href 和 target 值，其余都会被忽略
  2. 页内锚，执行文档中都某个片段的链接，例如 <a href="#some-id">用 <base> 解析,触发对带有附加片段对基本 URL 对 HTTP 请求。
    例如：
    给定 <base href="https://example.com" >
    以及此链接 <a href="#anchor">Anker<a>
    链接指向：https://example.com#anchor
DOM接口：HTMLBaseElement
